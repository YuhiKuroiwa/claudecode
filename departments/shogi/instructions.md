# 将棋大会情報部門

## 目的
関東圏で開催される対人(対面)将棋大会・イベントの情報を調査し、一覧化する。データサイエンス系コンテスト(contests部門)とは別の、個人の趣味・参加目的の部門。

## 対象範囲
2種類の情報を対象とする。

1. **大会(単発イベント)**: 将棋道場・日本将棋連盟支部・地域サークル等が主催する対人将棋大会
2. **定常的に対局できる場**: 将棋カフェ、将棋道場の定例会・例会(例: 毎週土曜開催、など定期開催されているもの)。単発の大会ではないが、実際に対局しに行ける場として同じくらい重要

共通条件:
- **地域**: 東京都、千葉県西部、埼玉県南部、神奈川県北部
  - 千葉県西部: 市川・船橋・松戸・柏・我孫子 など
  - 埼玉県南部: さいたま・川口・越谷・草加・戸田 など
  - 神奈川県北部: 相模原・厚木・大和・川崎北部 など
  (厳密でなくてよい、目安として判断する)
- **対象棋力**: 一般男性、級位者〜初段程度が参加しやすいもの(誰でも参加可のものを含む)。小学生・中学生限定、女流限定、高段者限定など明らかに対象外のものは除外する
- オンライン対局のみのもの(将棋ウォーズ大会等)は対象外
- 開催日がすでに過去のもの(調査日より前)は対象外

**除外したものはレポートに一切書かない。** 「小学生限定だから除外」「過去開催だから除外」のような除外理由の列挙・参考情報としての記載も不要。該当するものが無ければ単に「見つかりませんでした」とだけ書く(下記フォーマット参照)。

## 調査方法
- **大会情報**は専門ブログ「将棋大会・イベント情報」の都道府県別ラベルページを主な情報源とする
  - 東京: https://sho-shogi.blogspot.com/search/label/東京
  - 千葉: https://sho-shogi.blogspot.com/search/label/千葉
  - 埼玉: https://sho-shogi.blogspot.com/search/label/埼玉
  - 神奈川: https://sho-shogi.blogspot.com/search/label/神奈川
  - 全体像の把握には首都圏ページも参考にする: https://sho-shogi.blogspot.com/search/label/首都圏
- **将棋カフェ・道場の定例会**は上記ブログには載っていないことが多いため、X(Twitter)検索やWeb検索で補足する(例:「将棋カフェ 東京」「将棋道場 定例会 土曜 埼玉」「将棋サロン 神奈川」)。個人が発信している小規模な情報も対象にしてよい
- 開催日(定例会の場合は開催曜日・頻度)・場所・対象(棋力条件、参加費など)・申込方法が分かる範囲で記載する
- 前回のレポート(`departments/shogi/reports/` 内の直近ファイル)があれば読み、そこに無い新しい情報を優先的に取り上げる。ただし定例会(毎週開催など)は変化が無くても、まだ一度も報告していなければ載せる
- 情報源のURLを必ず記録する。個人ブログ・SNS等の非公式情報しか無い場合はその旨を明記する

## 出力先
`departments/shogi/reports/YYYY-MM/YYYY-MM-DD.md`(月ごとにフォルダを分け、その中に1日1ファイル)

## 出力フォーマット
- シンプルな箇条書き(概要/背景のような分割はしない)
- 通し番号を付ける
- 「大会」と「定例会・将棋カフェ」を見出しで分ける

```md
# 将棋大会情報レポート - YYYY-MM-DD

## 大会
1. [大会名](URL)
   開催日: YYYY-MM-DD / 場所: ◯◯(東京都◯◯区) / 対象: 級位者〜初段、誰でも参加可 / 参加費: ◯◯円
   申込: ◯◯から / 情報源: [◯◯道場公式サイト](URL)

## 定例会・将棋カフェ
2. [◯◯将棋道場 定例会](URL)
   開催頻度: 毎週土曜 / 場所: ◯◯(埼玉県◯◯市) / 対象: 誰でも参加可 / 参加費: ◯◯円
   情報源: [X投稿](URL)(非公式情報)
```

該当情報が見つからない場合:
```md
# 将棋大会情報レポート - YYYY-MM-DD

対象地域・対象棋力に合う情報は見つかりませんでした。
```

## 実行方法
当面は自動化せず、必要な時に手動で依頼する(この部門を毎朝の自動ルーティンには含めない)。

## 優先度・件数の目安
- 件数の上限は設けない。無理に数を揃えず、実際に見つかった大会のみ記載する
