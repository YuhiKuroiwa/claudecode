# ITニュース調査レポート - 2026-07-06

## 1. TeslaがAIツール利用に週200ドルの支出上限を導入(xAI/Grokは対象外)

**概要**

Teslaが7月6日より、Anthropic・OpenAI・Google製AIツールへの従業員個人利用支出を週200ドルに制限する社内方針を導入した。上限超過には承認が必要。一方、Elon Musk率いるxAIのGrok(ベータ版)は対象外。

**背景**

一部のソフトウェアエンジニアが週数千ドル相当のトークンを消費しており、コスト管理上の課題となっていた。過去半年、Teslaは全社的なAI利用の標準化(承認モデル・セキュリティポリシー)を進めており、今回はその延長で支出ガードレールを追加した形。数か月前まで全社的にAI利用を奨励していたところからの大きな方針転換となる。

**影響・所感**

企業内AIエージェント利用のコスト急増が一般的な経営課題になりつつあることを示す事例。xAIだけが対象外という点は、MuskグループのAI製品優遇として注目されている。

**情報源**

[The Information](https://www.theinformation.com/articles/tesla-caps-employee-ai-spend-200-per-week-adoption-push) / [Electrek](https://electrek.co/2026/07/02/tesla-caps-employee-ai-spending-200-week/)

## 2. Microsoft SharePointの重大脆弱性(CVE-2026-45659)が実際に悪用、CISAが緊急対応要求

**概要**

Microsoft SharePoint Serverのリモートコード実行脆弱性(CVSS 8.8)がCISAの既知悪用脆弱性(KEV)カタログに追加された。米連邦機関には7月4日までの修正適用が求められた。

**背景**

攻撃者による実際の悪用が確認されたための緊急対応。SharePointは多くの企業・政府機関で利用されており影響範囲が広い。

**影響・所感**

企業・組織は至急パッチ適用状況を確認する必要がある。継続的な標的型攻撃のリスクが高い。

**情報源**

[The Hacker News](https://thehackernews.com/2026/07/sharepoint-rce-cve-2026-45659-added-to.html)

## 3. Linuxカーネルの新たな深刻な脆弱性「Bad Epoll」(CVE-2026-46242)発見

**概要**

一般ユーザー権限から特別なアクセスなしにroot権限を奪取できるLinuxカーネルの脆弱性「Bad Epoll」が発見された。Linuxデスクトップ・サーバーに加えAndroidにも影響。

**背景**

epoll関連の実装欠陥に起因するとみられ、権限昇格の典型的な深刻度の高いパターン。

**影響・所感**

サーバー管理者・Androidベンダーは早急なパッチ適用状況の確認が必要。影響範囲の広さから今後詳細な悪用実証(PoC)の公開にも注意。

**情報源**

[The Hacker News関連報道](https://thehackernews.com/)

## 4. Microsoft、OpenAIとの提携条件を再編—収益シェア支払いを終了

**概要**

MicrosoftはOpenAIへの収益シェア支払いを終了する一方、OpenAIは2030年までMicrosoftへの支払いを継続する形にパートナーシップを再構築したと報じられた。GoogleはAnthropicへの投資拡大も伝えられている。

**背景**

生成AI大手同士の力学変化を反映。Microsoftは自社モデル開発を進めOpenAI依存を下げる動きを継続しており、Googleは複数のAI企業に分散投資する戦略を強めている。

**影響・所感**

大手クラウド・AI企業間の提携構造の変化は、今後のAIインフラ投資や競争構図に大きく影響する可能性がある。

**情報源**

[The Information](https://www.theinformation.com/newsletters/applied-ai/google-microsoft-team-beat-back-anthropic-openai) / [Fortune](https://fortune.com/2026/07/02/sam-altman-new-world-order-ai-openai-google-anthropic/)

## 6. AI半導体株が急伸、世界チップ市場は1.3兆ドル規模の見通し

**概要**

NVIDIAはAI GPU市場でシェア70〜80%を維持する一方、AMDのMI300シリーズがハイパースケーラー顧客の間でシェアを伸ばしている。Samsung・SK hynixは半導体クラスター・AIデータセンター向けに今後10年で1兆ドル超の投資を計画。

**影響・所感**

AI投資ブームが半導体セクター全体の株式市場評価を押し上げている構図が継続。メモリコスト高騰がスマートフォン等の最終製品価格に転嫁される可能性も指摘されている。

**情報源**

[Distill Intelligence Weekly Briefing](https://www.distillintelligence.com/briefings/semiconductors-ai-chips-2026-07-03)
